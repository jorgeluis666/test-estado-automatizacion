# Checklist de Automatizacion

Prototipo web de la Suite de Optimizacion Digital para diagnosticar el estado de automatizacion de una empresa.

## Que permite hacer

- Revisar ideas de automatizacion por area.
- Marcar automatizaciones que la empresa ya tiene.
- Marcar automatizaciones que quisiera desarrollar.
- Calcular estado actual y estado potencial.
- Exportar un resumen del diagnostico.

## Uso

Abre `index.html` en el navegador o sirve la carpeta con cualquier servidor estatico.
